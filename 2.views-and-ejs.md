## Views and EJS

EJS - embedded java
`npm i ejs`

.ccs files go in 'public' dir,
.html files go in 'views' dir

```js
app.use(express.static('public'))

app.set(''view engine', 'ejs');

app.get('/about', (req, res, next) => {
  res.render('about', someObject);
});
```
